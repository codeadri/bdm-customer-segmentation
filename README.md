# Customer Segmentation for a UK Online Retailer

This repository contains the analysis code for an individual academic project submitted as part of the **Business Data Management** course under the IIT Madras Online Degree Programme.

## Project Overview
The objective of this project is to improve marketing efficiency for a UK-based online retailer through customer segmentation using transactional data. The analysis applies the RFM (Recency–Frequency–Monetary) framework and K-Means clustering to identify distinct customer groups and derive actionable marketing insights.

## Dataset
The dataset used in this project is the **Online Retail Dataset**, obtained from the UCI Machine Learning Repository:
https://archive.ics.uci.edu/dataset/352/online+retail

The data represents real transactional records from December 2010 to December 2011.

## Repository Contents
- `BDM.ipynb`: Final reproducible analysis notebook
- Supporting cleaned data files used in the analysis

## Reproducibility
The notebook is fully reproducible and can be executed top-to-bottom to regenerate all results, tables, and figures presented in the accompanying project report.
